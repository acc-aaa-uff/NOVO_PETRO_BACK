Versão do Python utilizada - 3.11.4
Versão do pip utilizada - 23.2.1

Crie um ambiente vitual ...

O ambiente utilizado chama-se NOVO_PETRO (você pode criar o seu).
O arquivo requirements.txt, foi gerado da seguinte forma: pip freeze > requirements.txt.
Instale todas as bibliotecas utilizadas neste projeto -> no ambiente virtual que criou.
Para isso faça: pip install -r requirements.txt.
Os dados de entrada foram colocados no diretório chamado DADOS.
Os dados de saída foram gravados no diretório chamado DADOS_SIMUL.

Um jogo foi criado para iniciá-los no conceito de Backend e Frontend
Para o jogo foi criada uma api em Flask armazenada no diretório chamado api
O Backend do jogo foi criado no diretório chamado backend - foi utilizado Python para esse projeto
Dentro do diretório api há um diretório chamado templates - nele se encontra o frontend do jogo. Aqui pode se melhorar
muito utilizando CSS e React por exemplo.

Foi criado também um arquivo Python chamado S_Al_Novo_Petro_Pre_proc_POO.py - este código deverá ser melhorado e
integrado na solução do simulador, ele permite que se acesse um diretório com os dados de entrada e disponibilize para o
usuário um menu de escolha, isso eleva a qualidade da solução e elimina restrições de uso. Mas também deve ser
melhorado e aprimorado para que possa ser integrado ao simulador.

Para executar o Game, entre no diretório do game, no meu caso o diretório \api --> neste diretório execute o comando:
python Game_PAI_Flask.py

Abra seu browser e digite: 127.0.0.1:5000
